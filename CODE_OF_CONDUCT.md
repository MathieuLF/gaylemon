# Code de conduite

Gaylémon accueille les contributions techniques, documentaires et visuelles dans un cadre respectueux.

## Comportements attendus

- communiquer de façon directe et constructive;
- critiquer le code ou la décision, jamais la personne;
- respecter la vie privée des joueurs et des opérateurs;
- documenter les risques et limites connus;
- accepter qu'un changement dangereux pour une instance en production soit refusé ou reporté.

## Comportements inacceptables

- harcèlement, intimidation ou propos discriminatoires;
- publication de secrets, sauvegardes ou identifiants sans autorisation;
- contournement volontaire des protections de production;
- contenu trompeur ou malveillant;
- attaques personnelles dans les issues et demandes de fusion.

## Application

Les mainteneurs peuvent modifier, masquer ou refuser un contenu qui enfreint ces règles. Un incident doit être signalé par un canal privé, sans exposer publiquement les personnes ou données concernées.
